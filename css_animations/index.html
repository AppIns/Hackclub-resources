CSS animations

If you haven't seen this, go watch it Css animations are easy. Inspect this page. There is no JavaScript file on this page.

Start by creating a new HTML file inside a new folder. It should look something like this:
<html>
  <head>
   <link href="styles.css" rel="stylesheet">
  </head>
  <body>
  </body>
</html>



I can not stress this enough,

If you want to make this your own, feel free to Google (or ask) for help

Now create a file called styles.css in the same folder, most of our work will be in here.
To start out, we should create out first animation. Please don't copy paste the code,
type it out. You really want to remember how to do this without looking every time. @keyframes animation_name {
 from{color:red;}
 to {color:blue;}
}
We load our page but nothing happens...
Well we need an element to change, so lets do that.
Add some element. Why not some text? Lets add a P element.
<p> Some text here </p>
Now lets change it in CSS p{
  animation-name: animation_name;
  animation-duration: 4s;
}

It should look something like this

If you want it to repeat several times, add something like

animation-iteration-count: infinite;
If you want to delay the animation by several seconds, use
animation-delay: 1s; There are many other things you can do with just animations, One major one
I left out is the ability to have more than just a 'to' and 'from' statement.
You can replace them with 0% and 100% and add other values in between.
If you want to learn more about them, see this website all about animations.

    • Ideas

    • Use :hover to get elements to change when you hover over them.
    • Make elements change size by editing their width and height properties.
    • Make elements do several changes at the same exact time (see top of this page).
    • Make elments spin by using something like "transform: rotate(180deg);" and "transform:roate(0deg);" to rotate the element.
